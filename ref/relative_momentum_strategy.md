## 1. 手法の背景・出典

### 学術的根拠

モメンタム投資の学術的根拠は Jegadeesh & Titman (1993) の研究に遡る。
過去3〜12ヶ月のリターンが高かった資産は、将来の同期間においても高いリターンを示す傾向がある（モメンタム効果）。

### Dual Momentum（Gary Antonacci）

Gary Antonacci が著書 "Dual Momentum Investing" (2014) で体系化した手法。
2つのモメンタムを組み合わせる。

- 相対モメンタム（Relative Momentum）: 複数資産の中でリターンが高い資産を選ぶ
- 絶対モメンタム（Absolute Momentum）: 選んだ資産がそもそも上昇トレンドにあるか確認する

Antonacci の研究では、ルックバック期間として12ヶ月が最も安定した成果を示すことを確認している。

### 日本のiDeCo向けアレンジ（さかつだより / 市原ブログ参照）

「12ヶ月移動平均 + 3-6-12ヶ月レラティブストレングス投資」として日本のiDeCoに適用された手法。
月末に以下の8資産クラスのモメンタムスコアを計算し、上位2〜3資産を選択する。

対象資産:
- 国内株式
- 先進国株式
- 新興国株式
- 国内債券
- 先進国債券
- 新興国債券
- 国内REIT（J-REIT）
- 海外REIT（G-REIT）

## 2. スコアリングロジック

### 旧ロジック（3-6-12平均）

```
スコア = (3ヶ月リターン + 6ヶ月リターン + 12ヶ月リターン) / 3
```

各リターンを単純平均する。参考文献の sakatsulife.com の元ロジック。

### 本プロジェクトの改良版（1-3-6-12加重平均）

```
スコア = (1M×1 + 3M×2 + 6M×3 + 12M×4) / 10 − 信託報酬(%/年)
```

改良ポイント:
- 長期リターンに重みをかけることで長期トレンドを重視
- 信託報酬を控除することでコスト高の商品に自動的に不利なスコアを与える

理論的根拠:
- 学術研究では6ヶ月・12ヶ月の予測力が高いため重みを大きくしている
- DCは非課税口座であり信託報酬コストの影響が長期で大きい

## 3. BUY/SELLシグナル判定（2段階フィルター）

### 条件① 相対モメンタムフィルター（スコア基準）

```
score > 0
```

スコアがプラスの商品のみBUY候補とする。
スコアがマイナスとは「加重平均リターンが信託報酬を下回っている状態」を意味し、実質的にコストを払いながら下落または横ばいであることを示す。

### 条件② 絶対モメンタムフィルター（12ヶ月MA基準）

```
現在基準価額 / 12ヶ月移動平均 > 1.0
```

「現在価格が過去12ヶ月の平均価格を上回っているか」を確認する。
これが下回っている場合は下降トレンドにあると判断し、いくらスコアが良くてもSELLとする。

Antonacci の絶対モメンタムに相当する部分。大きな下落局面（リーマンショック・コロナショック等）で
リスク資産から退避する役割を果たす。

代替指標: 12ヶ月分のNAV履歴がない場合は「12ヶ月リターン > 0」をプロキシとして使用。

### 条件③ 短期トレンド確認（加速フィルター）

```
3ヶ月移動平均(MA3) > 6ヶ月移動平均(MA6)
```

短期の価格移動平均が中期を上回っているか確認する。
これが下回っている場合は「上昇が鈍化・反転の兆候あり」として保守的にSELLとする。

計算方法: 月次リターン履歴から過去のNAVを逆算し、MA3とMA6を算出する。

## 4. ポートフォリオ構築

### 掛金割当変更（毎月実行）

BUY候補の上位TOP_N本（デフォルト3本）を選定し、均等配分（33%ずつ）またはスコア比例配分で割り当てる。

均等配分の利点:
- シンプルで管理しやすい
- 特定商品への過度な集中を防ぐ

スコア比例配分の利点:
- スコア差が大きい場合に上位集中が可能
- モメンタムの強さに応じたダイナミックな配分

### BUY候補ゼロの場合

全BUY候補がゼロの場合（市場全体の下落局面）は、掛金100%を元本確保型商品に振り向ける。

## 5. スイッチング（残高移動）ルール

DCは非課税のためスイッチングコストは信託財産留保額のみ。
500〜700万円規模の残高では積立変更のみでは修正に数年かかるため、積極的にスイッチングを活用する。

### ケースA: 緊急離脱（高優先度）

```
条件: SELLシグナルが2ヶ月連続で点灯している資産
対応: 該当資産の残高を全額スイッチング → 現在のBUY上位商品へ
```

理由: 一時的なノイズではなくトレンド転換が確認されたと判断する。
2018年の国内株式暴落時にこのルールが有効に機能した（バックテスト結果より）。

### ケースB: 通常リバランス（低優先度）

```
条件: 目標配分との乖離が15%以上 かつ 3ヶ月以上経過
対応: 乖離分のみスイッチング → 目標配分に近づける
```

15%閾値の根拠:
- 信託財産留保額が0.3%ある商品は頻繁なスイッチングでコストがかさむ
- 小さな乖離へのスイッチングを防いでコスト効率を維持する

## 6. 手法の特性と注意点

### 強み

- 大きな下落局面を避けられる可能性が高い（絶対モメンタムフィルターの効果）
- 上昇相場では高リターン資産に集中できる
- ルールベースで感情的な判断を排除できる
- 月1回の確認で運用できる（低メンテナンスコスト）

### 弱点

- トレンド転換直後（シグナルの「だまし」）では損失が出る
- 横ばい相場では頻繁なシグナル変化によりコストがかさむ
- モメンタムは市場が効率的でない場合に機能するため、将来の有効性は保証されない
- バックテストの過学習リスク（成功した時期のデータで評価しているため）

### iDeCo固有の制約

- スイッチングの申請から反映まで数営業日のラグがある
- 月初に判定し月中に実行する想定で運用する
- 信託財産留保額がある商品（0.3%等）は頻繁なスイッチングを避ける
- 商品ラインナップは証券会社ごとに異なるため商品コードの定期確認が必要

## 7. バックテスト結果（2018〜2019年、本プロジェクトの参考記録）

| 時期 | 市場状況 | 主なBUY候補 | 特記事項 |
|---|---|---|---|
| 2018-07 | 上昇相場 | J-REIT・海外小型株・Yjam+ | |
| 2018-08 | 安定 | 海外小型株・海外REIT・J-REIT | |
| 2018-10 | 上昇ピーク | TOPIX100・海外小型株・海外株 | |
| 2018-11 | 暴落開始 | J-REIT・海外REIT | 国内株は早期SELL |
| 2018-12 | 暴落継続 | J-REIT・海外REIT | REITが防御的に機能 |

- 元本保証への完全退避は発動しなかった（12ヶ月MAフィルターで緩衝）
- REITと債券が暴落局面のバッファとして機能した
- 国内株式の2ヶ月連続SELLが早期発動し、ケースAが有効に機能した
